# DS2026 Practicum

Учебные проекты курса yandex practicum Data Science 2026.

Репозиторий содержит выполненные ноутбуки по классическим задачам анализа данных и машинного обучения: EDA, классификация, регрессия, ансамбли, бустинг, кластеризация и работа с признаками.

## Структура

| Папка | Тема | Содержимое |
|---|---|---|
| `projects/module-3-logreg` | Логистическая регрессия | churn-классификация |
| `projects/module-3-svc` | SVM / SVC | классификация и сравнение моделей |
| `projects/module-3-multiclass` | Multiclass classification | многоклассовая классификация |
| `projects/module-4-decision-tree` | Decision Tree / KNN | регрессия и подбор модели |
| `projects/module-4-random-forest` | Random Forest | поведенческий скоринг |
| `projects/module-4-boosting` | Gradient Boosting | бустинг и сравнение моделей |
| `projects/module-4-urbanstay` | Финальный проект модуля 4 | прогноз отмен бронирований |
| `projects/module-5-unsupervised` | Unsupervised learning | кластеризация и риск задержек доставки |
